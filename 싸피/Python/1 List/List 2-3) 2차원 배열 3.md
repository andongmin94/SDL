### 2차원 배열의 접근

#### 델타를 이용한 2차 배열 탐색

* 2차 배열의 한 좌표에서 4방향의 인접 배열 요소를 탐색하는 방법

```
arr[0...N-1][0...N-1] # N x N 배열
di[] <- [0, 1, 0, -1]
dj[] <- [1, 0, -1, 0]

for i : 0 > N-1
    for j : 0 -> N-1 :
        for k in range(4) :
            ni <- i + di[k]
            nj <- j + dj[k]
            if 0 <= ni < N and 0 <= nj < N # 유효한 인덱스면
                f(arr[ni][nj])
```

```python
'''
3
1 2 3
4 5 6
7 8 9
'''
di = [0, 1, 0, -1]
dj = [1, 0, -1, 0]
N = int(input())
arr = [list(map(int,input().split())) for _ in range(N)]
# arr = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

max_v = 0 # 모든 원소가 0 이상이라면...
for i in range(N): # 모든 원소 arr[i][j]에 대해...
    for j in range(N):
        # arr[i][j]를 중심으로 사방탐색하여 s에 더함.
        s = arr[i][j]
        for k in range(4):
            ni, nj = i + di[k], j + dj[k]
            if 0 <= ni < N and 0 <= nj < N: # 배열을 벗어나지 않으면
                s += arr[ni][nj]
        # 여기까지 주변 원소를 포함한 합
        if max_v < s:
            max_v = s
```
