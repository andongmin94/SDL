### 문자열 암호화
#### 시저 암호
* 줄리어스 시저가 사용했다고 하는 암호
* 시저는 100년 경에 로마에서 활약한 장군임.
* 시저 암호에서는 평문에서 사용되고 있는 알파벳을 일정한 문자수만큼 평행이동시켜서 암호화를 행한다.

#### ex) 1만큼 평행했다치면
* 평문 Y -> 암호문 Z
* 1만큼 평행했을 때 1을 키값이라 함.
* 수신자 이외의 사람이 암호문을 보고 메시지를 맞출 수 없음.
* 시저암호를 해독할 수 없을까?

#### 문자 변환표로 암호화 하기도 함.

* 복호화 하기 위해서는 모든 키의 조합이 필요하다.
