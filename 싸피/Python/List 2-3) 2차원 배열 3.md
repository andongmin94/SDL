### 2차원 배열의 접근

#### 델타를 이용한 2차 배열 탐색

* 2차 배열의 한 좌표에서 4방향의 인접 배열 요소를 탐색하는 방법

```
arr[0...N-1][0...N-1] # N x N 배열
di[] <- [0, 1, 0, -1]
dj[] <- [1, 0, -1, 0]

for i : 0 > N-1
    for j : 0 -> N-1 :
        for k in range(4) :
            ni <- i + di[k]
            nj <- j + dj[k]
            if 0 <= ni < N and 0 <= nj < N # 유효한 인덱스면
                f(arr[ni][nj])
```

```python
'''
3
1 2 3
4 5 6
7 8 9
'''
di = [0, 1, 0, -1]
dj = [1, 0, -1, 0]
N = int(input())
arr = [list(map(int,input().split())) for _ in range(N)]
# arr = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
```
